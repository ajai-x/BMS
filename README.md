# Battery-Management-System
My final year project - Advanced Battery Management System with predictive analysis and Active cell balancing
  1. Developed an ESP32-based smart Battery Management System (BMS) for real-time monitoring of voltage, current, and temperature in a multi-cell battery pack.
  2. Integrated ADS1115 ADC, INA219 current sensor, and DS18B20 temperature sensor to collect accurate battery parameters and ensure safety monitoring.
  3. Implemented machine learning models (Random Forest) to predict thermal runaway risk and estimate battery Remaining Useful Life (RUL).
  4. Designed active cell balancing circuitry and an IoT dashboard using Plotly/Dash for battery health visualization and real-time alerts.
